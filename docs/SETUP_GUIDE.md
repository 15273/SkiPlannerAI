## שלב 1 — יצירת Database ב-pgAdmin ✅/⬜
> **את עושה** — פתיחת pgAdmin ויצירת הbases

1. פתחי את **pgAdmin** (הותקן יחד עם PostgreSQL)
2. בצד שמאל — לחצי ימנית על **Databases** ← **Create** ← **Database**
3. בשדה *Name* כתבי: `skiplanner`
4. לחצי **Save**
5. לחצי על `skiplanner` ← **Query Tool** (כפתור למעלה)
6. הדביקי את הפקודה הזו ולחצי ▶️:
   ```sql
   CREATE EXTENSION IF NOT EXISTS postgis;
   ```
7. אמור לכתוב `Query returned successfully`
