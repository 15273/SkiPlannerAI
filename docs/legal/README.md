# Legal Docs

This folder contains skiMate's legal and compliance documents.

| Document | Description |
|----------|-------------|
| [location-privacy-policy.md](location-privacy-policy.md) | How skiMate collects, uses, stores, and shares location data (GPS, IP-inferred, and user-entered origin city/airport). Covers GDPR, CCPA, COPPA, and OSM attribution. |

## Adding new documents

Place new legal documents in this folder and add a row to the table above. Use kebab-case filenames (e.g. `terms-of-service.md`, `cookie-policy.md`).
